# E-commerce Customer Analysis

## Project Overview
This project analyzes customer purchase data using Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Column Charts

## Analysis Performed
- Monthly Orders Analysis
- Customer purchase trends

## Insights
The analysis shows that customer orders were higher in May compared to June. Pivot tables and charts help visualize order trends and customer activity.
